FxWidget provides a widget for displaying official ECB exchange rates for the Euro.

== Description ==

Functionality:

FxWidget provides a widget for displaying official ECB exchange rates for the Euro. Any number of 33 currencies can be selected for display in the widget's options. Exchange rates are loaded from the official ECB site which updates the rates daily at around 3.00 p.m. CET.

Localizations:

de_DE
en_US

== Installation ==

1. Download the plugin archive and unzip it.
2. Copy the resulting folder to the folder wp-content/plugins/ of your Wordpress installation.
3. Activate the plugin.
4. In Appearance -> Widgets drag the widget FxWidget to the desired spot in the sidebar.
5. In the widget's options select the currencies you want the widget to display.
